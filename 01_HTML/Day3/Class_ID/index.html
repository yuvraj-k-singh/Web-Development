Class and ID

What is class?

  • A class is a way to define a group of elements that share the same styling or behavior.
  • It can be applied to multiple elements within a webpage.
  • Class name can be same for multiple elements.
  • It is selected by using . (dot) notation in CSS.

What is ID?

  • An ID is a unique identifier for an element within a webpage. It can only be used once per page.
  • ID name must be unique for each element. It cannot be same for different elements on the same page.
  • It is selected by using # (hash) notation in CSS.
