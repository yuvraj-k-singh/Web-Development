Image Tag

The image tag is used to embed an image in an HTML page.

An image is a visual representation of a person, object, or scene, which can be captured through photography or created via drawing, painting, or digital software. While the term is often used interchangeably with "photo" or "picture," there are technical differences: a photo is specifically an image captured by a camera using light, whereas an image is a broader category that includes digital renders and mental visualizations

Common Types of Images

  • Photographs: Real-life moments captured by cameras or mobile phones.
  • Digital Art: Images created using digital tools and software.
  • Illustrations: Artwork created using graphic design software.
