External CSS

What is External CSS?

External CSS is a method of adding CSS styles to an HTML document by linking to an external .css file.

Advantages of External CSS

  • External CSS allows you to define styles for multiple pages within a single file.

  • It improves page loading performance by reducing the amount of code in the HTML file.

  • It provides better maintainability and organization of styles across a website.
